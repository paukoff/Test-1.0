Нужно в довольно сжатые сроки сделать сайт с простой админкй для управления контентом. Сначала нужно сверстать все 5 страничек адаптивно на Bootstrap 3.0, затем реализовать простую админку, ну и связать все это дело между собой.

Исходники дизайна в .psd. Есть вариант для самого большого разрешения и для самого маленького. Адаптивность простейшая. Также есть краткое ТЗ для админки, каким контентом можно управлять. Дизайн админки не важен, важно чтобы она работала и контент менялся.
